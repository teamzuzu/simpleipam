# SimpleIPAM - Simple IP address management

Cut down version of the original in docker etc


## Using
* Codeigniter 3.1.5 
* Bootstrap v3.3.7
* jQuery v3.2.1
* DataTables 1.10.7

## Configuration


### Change Per Page
Edit $config['per_page'] in controllers/Networks.php and controllers/Hosts.php  
For example, change  "5" to "300"
